# Documentation

Supporting material for the test strategy.

- **`Berkeley_QA_Test_Strategy.docx`** — the full QA test strategy: guiding philosophy, the layered test model for an API consumer, test types, risk-weighted coverage per suite, tooling, black-box techniques, and white/grey-box recommendations.
- **`Berkeley_QA_Strategy_Deck.pptx`** — presentation version of the same material, including a dedicated load & performance section. Speaker notes are attached to each slide.

Both are intended to be read alongside the Playwright suite in this repository: the documents explain the *approach*, the code demonstrates it.
